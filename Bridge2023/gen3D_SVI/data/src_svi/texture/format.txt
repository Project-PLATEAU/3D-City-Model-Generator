テクスチャの貼られた建物の情報は建物ブロックごとに記述され、
建物ブロックはlod0, lod1の２つのブロックで構成されます。

・建物ブロック
id bldg_XXXXXXXXX
[lod0 ブロック]
[lod1 ブロック]


・lod0ブロック
1行目は固定で、２行目に形状を記載します。
polylineで建物外形を記述し、線分列になっています。位置はWGS84で記載します。

lod0 roof
pos point_num Lat0 Lon0 Alt0 Lat1 Lon1 Alt1 ....


・lod1ブロック
１行目にサーフェイスの数を示し、サーフェイスごとに頂点位置と三角形要素の記述を２行で行います。
頂点位置はlod0と同じ形式です。
三角形要素は三角形の頂点番号の組を要素数だけ列挙しており、
その後ろに対応するテクスチャの位置を画像の(u,v)座標で列挙しています。
ただし、テクスチャのない要素の(u,v)座標はNoneとしています。
画像ファイル名は建物の番号と対応しています。

lod1 surface_num
pos point_num Lat0 Lon0 Alt0 Lat1 Lon1 Alt1 ...
face face_num id0-0 id0-1 id0-2 id1-0 ... u0-0 v0-0 u0-1 v0-1 u0-2 v0-2 u1-0 v1-0 ...
